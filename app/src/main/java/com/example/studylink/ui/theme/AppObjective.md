Proposta de Projeto: StudyLink

- Uma app onde alunos guardam e fazem (ou não) a partilha de anotações com outros alunos.
  - Usar o padrão MVVM (Model-View-ViewModel), que separa a lógica de apresentação (Views) da lógica de dados (ViewModels);

Objetivos: (Requisitos obrigatorios)
- Ligação com o Firebase;
- Login;
- Registo;
- Perfil de Utilizador (Pagina onde cria ou edita nome e username, tem info de quantos e quais itens criou, e da data de criacao da conta);
- Lista de itens: (Necessario criar primeiro perfil)
  - Criar item (cria item com titulo, username, data, ultimavez editado, conteudo, tags);
  - Atualizar item;
  - Remover item.
