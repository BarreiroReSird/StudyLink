# StudyLink

## Nota

Este projeto foi desenvolvido no âmbito de um trabalho académico. O código-fonte está disponível no GitHub.

## Repositório

[https://github.com/BarreiroReSird/StudyLink](https://github.com/BarreiroReSird/StudyLink)

## Descrição

O StudyLink é uma aplicação Android que permite aos utilizadores gerir notas de estudo. A aplicação inclui funcionalidades de registo, autenticação, gestão de perfil de utilizador e a criação, visualização, edição e eliminação de notas.

## Funcionalidades

*   Registo e Login de Utilizadores
*   Gestão de Perfil
*   Criar, ver, editar e apagar notas

## Tecnologias

*   Kotlin
*   Jetpack Compose
*   Firebase Authentication
*   Firebase Firestore
